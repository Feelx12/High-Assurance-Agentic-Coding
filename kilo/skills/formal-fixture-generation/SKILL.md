# Skill: Formal Fixture Generation

Use after Alloy predicates/checks produce valid instances or counterexamples.

## Goal

Convert formal scenarios into executable verification artifacts.

## Workflow

Alloy predicate
→ generate instance
→ export JSON
→ create deterministic fixture
→ generate unit test
→ generate integration test if needed
→ link fixture/test to requirement ID

## Formal Fixture Summary
