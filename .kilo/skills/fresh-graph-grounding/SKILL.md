# Skill: Fresh Graph Grounding

Use at the start of Class 3 and Class 2 workflows.

## Goal

Ensure planning uses the current codebase, not stale architecture knowledge.

## Steps

1. Regenerate Joern/CPG graph.
2. Save graph snapshot ID.
3. Compare current graph to previous approved graph.
4. Identify changed symbols, dependencies, APIs, tests, data flows, and call paths.
5. Determine impacted architecture cone.
6. Output Structural Truth.
